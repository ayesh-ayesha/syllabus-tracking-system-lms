### ğŸš€ Overview
**A Digital Syllabus Tracker that bridges the gap between Student clarity and Admin control.**

This application solves the common problem of **"Syllabus Confusion"** in coaching centers. Instead of overwhelming students with content, it provides a clean, interactive roadmap (checklist) of topics. Students track their own progress, while Admins retain full oversight and access control.

---

### ğŸ“± Key Features

#### ğŸ”¹ For Students (The Roadmap)
* **Visual Syllabus:** Interactive checkboxes for course outlines. Students can mark topics as done/undone non-sequentially.
* **Real-Time Progress:** Automatic calculation of course completion (e.g., "40% Done") visualized with progress bars.
* **My Courses:** A dashboard of all enrolled courses.

#### ğŸ”¹ For Admins (The Control Center)
* **Course Management:** Create courses and define detailed topic outlines.
* **Student Monitoring:** View the exact progress bar of any specific student to verify their work.
* **User Management:** Fast search and filtering of students.
* **Access Control:** One-click **Active/Inactive** toggle to instantly grant or revoke app access for students.
